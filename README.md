# Web-Reservas-Universitarias
Repositorio para P1

Este proyecto es una aplicación web para la reserva de citas. A continuación, se presentan los pasos para ejecutar el proyecto localmente.

## Pasos para ejecutar el proyecto

1. Clonar el repositorio:

   ```bash
   git clone https://github.com/jdj40211/Web-Reservas-Universitarias.git

2. Navegar a la carpeta del proyecto
   
   cd WebReservaCitas

3. Ejecutar migraciones

   python manage.py migrate

4. Correr el servidor

   python manage.py runserver
